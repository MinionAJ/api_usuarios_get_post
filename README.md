# 🚀 API de Usuários com FastAPI

Projeto desenvolvido com foco em aprendizado de desenvolvimento backend utilizando Python.
A aplicação consiste em uma API REST para gerenciamento de usuários, incluindo operações básicas de cadastro e listagem.

---

## 🧠 Objetivo

Criar uma base sólida em desenvolvimento backend, abordando conceitos como:

* Criação de APIs REST
* Integração com banco de dados
* Estruturação de projeto em Python
* Uso de frameworks modernos

---

## 🛠️ Tecnologias utilizadas

* Python
* FastAPI
* Uvicorn
* SQLAlchemy
* SQLite

---

---

## ▶️ Como executar o projeto

### 1. Clonar o repositório

```bash
git clone <url-do-repositorio>
```

### 2. Acessar a pasta do projeto

```bash
cd projeto_api
```

### 3. Instalar dependências

```bash
pip install fastapi uvicorn sqlalchemy
```

### 4. Executar o servidor

```bash
uvicorn src.main:app --reload
```
